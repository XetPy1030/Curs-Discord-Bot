!list - показывает что можно написать, чтобы бот ответил
!set - ставит текст для ответа, пример: !set re=1(для лучшей проверки); text(текст, на который бот реагирует)=jasdgwgdgd; repl(ответ бота)=awdgvwad;
Дополнительно для !set text re - <@!\d{18}> - для упоминания, и тд
Дополнительно для !set repl - [{bot}] упомянет бота, [{author}] - упомянет автора смс, [{(число от 0)}] - берет упоминание от текста смс
!enable - включить ответки бота
!enable false - отключить ответки бота
!delete num=(число); = удалить (число) элемент ответа для бота
!delete text=(строка); = удалить элемент, который равен (строка) ответа для бота
